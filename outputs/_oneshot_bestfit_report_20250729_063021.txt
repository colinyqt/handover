=== Clause: 1.20.4 Multi-Function Electronic Meters ===

Requirements (condensed):
- Frequency range: ±0.5%
- Harmonic order limits: ±1% up to 31st harmonic order
- Accuracy class: Class 0.5
- Memory / logging capacity: Sufficient built-in memory capable of maintaining all data collected for a minimum of 36 months
- Current ranges: ±0.5%
- Voltage ranges: ±0.5%
- Communication interface: RS485, RJ45 or similar socket and plug as Approved
